# BroadLinkESP-Alexa
BoradLink Power Muti Outlet Remote Controll by Amazon Alexa

ESP8266 을 이용한 2개의 전원제어용 릴레이와 Amazon Alexa 의 인터페이스와 
BoardLint 멀티탭 사이의 게이트웨이로 전원 ON/OFF 를 음성으로 제어하는 프로그램.
